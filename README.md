# Static Blog Setup
A website solely for yourself to use as a blog, built with a helper server.

# What 
This is a self-site built with HTML, CSS, and JavaScript that is not hosted outside the local machine.

## Viewing the Blog
Open `index.html` in your browser (if you used IntelliJ for this project, Firefox might struggle with opening this file.)

### Requirements
- `Node.js` is required to run the `save-server.js` file, though it is not required for viewing the website itself.

## Backups
To back up the whole blog, copy the `posts.js` (or the entire folder) to an external drive, cloud storage, etc.

# Why
I wanted a simple place for writing a blog which the posts only belong to one's self. It's also just fun to have your own website that you can look at.

# Notes & Constraints
- This project is for local use only. It's not intended to be deployed in a public server.
- `save-server.js` should be run while adding a post.
